BDWM BBS Crawler

Course Project of Java Programming(2013 Spring)

A highly stylized thread viewer for PKU BBS. A dedicated crawler pulls down all the threads of user-specified boards using java multi-threading. Texts are then scanned and processed to calculate features of the current board. A overview is shown and then user can click to see details of the thread including the social network of each forum section.
