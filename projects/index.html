Mini Football Manager System

2013-2014 Spring – Introduction to Database Systems(Honor Track)

A mini online football manager system using Django. It supports two kinds of users, administraor and manager. Both kinds of users can query information of football players and coaches. Administrators can insert and delete the records while managers can make, accept or reject offers on players and coaches.

Cardiac Surgery Effect Prediction

2012-2013 Summer – Machine Learning

Tried some methods to solve three big problems in the medical data from the hospital: how to deal with the missing values, which features we should use (there are up to 73 features), and how to classify the patients. I tried KNN Imputation (KNNI) to finish data augmentation and tested ANN and KNN to do the feature selection.

BDWM BBS Crawler

2012-2013 Spring – Java Programming

A highly stylized thread viewer for PKU BBS. A dedicated crawler pulls down all the threads of user-specified boards using java multi-threading. Texts are then scanned and processed to calculate features of the current board. A overview is shown and then user can click to see details of the thread including the social network of each forum section.

Keywords Retrieve on Weibo

2012-2013 Fall – Data Structure and Algorithms(A)(Honor Track)

A real-time keywords retrieve tool for Sina Weibo. The composition modules are crawler, keywords extracter, index and user-interface. Through this tool, users can determine which keywords to occur or not in the tweets he/she follows, after the tool builds an inverted index.

Link game

2012-2013 Fall – Practice of Data Structure and Algorithm

Link game is an classic casual game. This is a link game with the interface of Plants vs Zombies and automatic solving mode.

Politicians’ Network in People’s Daily

2011-2012 Fall – Introduction to Computing (A) (Honor Track)

Extracting the entities of offices and names in People’s Daily from 1970 to 1979, then building social networks between politicians to see the change of power network of China in that decade.
